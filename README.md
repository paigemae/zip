# Zip Tools

Simple Go programs for zipping and unzipping files.

## Build

```bash
task build
```

## Usage

### Zip
```bash
./bin/zip output.zip file1.txt file2.txt
./bin/zip output.zip directory/
./bin/zip output.zip file1.txt directory/ file2.txt
```

### Unzip
```bash
./bin/unzip archive.zip
```

## Test
```bash
task test
```
